Formatting Code Samples with Javascript

Edit

A simple way to format code samples and add a copy button

The look and feel of the code samples comes from Google's prettify Javascript.

The Copy button in the code samples below is styled from Bootstrap's CSSs. The only thing that I couldn't get to work was the tooltip on hover the way the official Bootstrap site has it. Would be cool to have that, but it's not the main thing right now. The main thing is that you can copy code multiple samples. I orginally chose another Javascript option for copying code snippets. At first, clipboard.js was difficult to understand, since I don't have a great Javascript understanding. First attempt using this JavaScript. This didn't work because, though it successfully copied the code sample to the clipboard, multiple samples on the same page didn't work; you could copy one, but not more than one.

The solution came with using clipboard.js, though that took a while to figure out.
